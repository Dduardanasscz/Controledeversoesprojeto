# ExpoFest 2026

## Sobre o projeto
O ExpoFest é um site de evento que apresenta informações sobre atrações, ingressos e programação.  
O objetivo do projeto é praticar desenvolvimento web e o uso de versionamento de código com Git e GitHub.

---

## Funcionalidades
- Página inicial do evento  
- Seção de atrações  
- Informações de ingressos  
- Layout responsivo com HTML e CSS
